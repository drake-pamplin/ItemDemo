# ItemDemo
Repo for the item demo project.

## Build Instructions
1. Navigate to "Code/".
2. Run "./gradlew clean build".
3. Navigate to "Code/app/build/distributions/".
4. Unzip the file "app.zip".
5. Navigate to "Code/app/build/distributions/app/app/bin/".
6. Run "app.bat".
